ul은 unordered list 의 약자이다.
사전적 의미는 중요하지 않다.
속성값은 3개, circle, disc, square 가 있으나, 이것 역시 속성값을 외울 필요는 없다. html5에서는 지원 하지 않기때문.
중요한 것은 레이아웃을 구성 할때, navigation, board, sitemap 등 목록 형태의 컨텐츠를 표현하는 중요한 레이아웃 블럭 이라는 점만 기억 하면 되겠다.

  1. 번호 있는 목록 1, 목록표시 대문자 알파벳
  2. 번호 있는 목록 2, 목록표시 소문자 알파벳
  3. 번호 있는 목록 3, 목록표시 대문자 로마숫자
  4. 번호 있는 목록 4, 목록표시 소문자 로마숫자
  5. 번호 있는 목록 5, 목록표시 아라비아 숫자

ol은 ordered list 의 약자이다.
사전적 의미는 중요하지 않다.
속성값은 4개, "A, a, I, i"가 있다.
중요한 것은 레이아웃을 구성 할때, 번호로 정렬 되어야 하는 목록 형태의 컨텐츠를 표현하는 레이아웃 블럭 이라는 점을 기억 하길 바란다.

HTML
Hyper Text Markup Language
XML
Extensible Markup Language
dl
definition list

dl 은 definition list 의 약자이다.
정의형 목록인데, 마찬가지로 사전적 의미는 중요하지 않다.
특별한 속성값은 없다.
중요한 것은 레이아웃을 구성할때, 3단 형태 박스의 리스트 를 만들어 낼 때 유용하게 사용하기도 한다. (웹표준에서 어긋남)
