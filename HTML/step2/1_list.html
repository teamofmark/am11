ul은 unordered list의 약자.
사전적 의미는 비중요.
속성값은 총 3개(circle, disc, square) 하지만 이것도 비중요.
html5 에서는 tag의 속성은 css로 넘기는 것을 권장.
중요한 것은 layout 구성시 navigation, board, sitemap, processmap.. 이런 목록 형태 content 구조로 사용 된다는 점.

ol은 ordered list의 약자.
사전적 의미는 비중요.
속성값(attr / attribute) 4개, "A, a, I, i".
중요한 것은 layout 구성시 순서가 있거나 번호로 정렬 되어야 하는 목록 형태 content에 사용된다는 점.

dl
definition list

dl은 definition list의 약자.
정의형 목록인데 사전적의미 비중요.
특별한 속성값은 없다.
중요한 것은 layout 구성시 3단 형태 boxList등의 내부중첩구조 표현에 적합하다.
