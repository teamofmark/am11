ul은 unordered list의 약자.
사전적의미는 비중요.
속성값(attr = attribute) 3개. = circle, disc, square. 외울필요없음.
중요한 점은 layout 구성시 navigation, board, sitemap 등의 비순차목록형태의 content을 구조화 하기 좋다.

ol은 ordered list의 약자.
사전적 의미 비중요.
대표적인 속성값은 4개(+@), "A, a, I, i"
layout 구성시 순번이 정해져 있거나 번호로 정렬 되어야 하는 목록 형태의 content를 구조화 하기 좋다.

dl
definition list

dl 은 definition list 의 약자이다.
정의형 목록인데, 마찬가지로 사전적 의미는 중요하지 않다.
특별한 속성값은 없다.
한개의 title(dt)을 기준으로 보완하는 내용의 data(dd)를 추가삽입할 때 유리하다.
dt와 dd의 갯수가 딱히 정해지지는 않았으나 dt x 1, dd x 여러개를 추천한다.
